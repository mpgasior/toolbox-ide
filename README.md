# toolbox-ide [![Container Image CI](https://github.com/mpgasior/toolbox-ide/actions/workflows/container-image-ci.yml/badge.svg)](https://github.com/mpgasior/toolbox-ide/actions/workflows/container-image-ci.yml)

This repository contains my personal development environment setup, primarily designed for [Fedora Silverblue](https://fedoraproject.org/silverblue/) using [Toolbox](https://github.com/containers/toolbox).

## Installation

To configure the environment, run the setup script:

```bash
sudo ./setup.sh
```
